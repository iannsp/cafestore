# Café Store

An AI information service for my favorite coffee shop 


Nesse projeto vou mapear passo a passo da experiencia de desenvolver um serviço de informação sobre produtos de uma loja que eu gosto, a [café Store](https://www.cafestore.com.br).

Os objetivos é ter um modo de pesquisar os produtos da loja por um canal diferente do site. Acho que o primeiro que penso é o terminal, depois um whatsapp.

Junto com isso eu quero ir escrevendo sobre como esta sendo feito. Aqui acho que vale frisar o **como esta sendo feito**. Ao invés de escrever como retrospectiva eu vou escrever enquanto estou fazendo. Isso vai criar vários caminhos além do certo/executado.  

As certezas são:
- Vai ter uma mistureba de coisas no "data crunch".
- O código final da aplicação vai ser escrito em Go.
